# NHS Prescribing Insights

## Overview
This project explores NHS prescribing data to uncover trends, patterns, and insights using Python-based data analytics techniques.

The project focuses on cleaning healthcare data, performing exploratory data analysis (EDA), and creating visualisations to communicate meaningful findings.

## Objectives
- Clean and prepare NHS prescribing datasets for analysis
- Explore prescribing trends and patterns
- Identify key insights from healthcare data
- Create clear data visualisations for communication

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Status
🚧 Work in progress

Currently focusing on data cleaning and exploratory analysis. Further insights and visualisations will be added as the project develops.
